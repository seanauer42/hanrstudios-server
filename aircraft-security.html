Aircraft Security

  1. Which of the following are NOT areas required to be inspected during the Aircraft Security Checks by maintenance?
    Flight Deck storage compartments
  2. What do the letters “NA” in a Serial Number position of the Security Check option, under the Aircraft Status tab of the DCC indicate?
    The panel is not installed.
  3. Where can you find the policy that describes the installation and documentation of Security Labels, and the completion of the Aircraft Security Check in the DCC Application within Workbench?
    AMDM 03-0-90
  4. When inspecting the lavatory, you notice that the security label on the shroud is missing, FedEx Aircraft Maintenance or an authorized vendor must re-inspect the compartment, apply a new Security Label and document its serial number in DCC.
    True
  5. What should you do if you find a suspicious item during the Aircraft Security Check?
    Contact Corporate Security, but do not remove or disturb the item.
  6. What is required, if there is no stock of Security Labels, or the application of a label would result in a delay?
    All of these.
  7. Who is responsible for the positive control of the numbered security labels after being issued to aircraft maintenance?
    Aircraft Maintenance Management
  8. All FedEx Express trunk aircraft with access doors or panels that may reasonably be opened with a screwdriver or Allen wrench, and which are accessible from the ground without using a ladder, must be inspected for contraband when performing an Aircraft Security Check.
    True
  9. If there is any evidence of tampering to a Security Label, what must be done?
    A FedEx Aircraft Maintenance Technician or an authorized vendor must re-inspect the compartment, apply a new Security Label, and document its serial number in DCC.
  10. When conducting the Aircraft Security Check, you are looking for contraband. What is considered to be contraband?
    All of these
